# Tic Tac Toe

## Install

```bash
$ composer install
```

## Tests

```bash
$ ./vendor/bin/phpunit
```

## Front

Pour un jeu rudimentaire et bien voir le fonctionnement

```bash
$ ./server.sh
```

Se rendre sur http://localhost:8080
http://localhost:8080?size=X (X est un nombre >= 3) pour changer la taille
